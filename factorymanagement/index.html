Owner Actions

Update Router

Change the PancakeSwap V2 router used for migration.

Update Fee Recipient

Address that accumulates platform fees (creation + trade).

Update Trade Fee

Fee on each buy/sell (in BPS, max 500 = 5 %). Default: 100 (1 %).

Update Creation Fee

Fee charged to launch a new token, denominated in USD (18-dec). Default: $1.

Update USDC Oracle Pair

Swap the TWAP oracle pair. isToken0 is derived on-chain. Resets TWAP state.

TWAP Max Age (Blocks)

Max blocks since last successful TWAP refresh before it's considered stale. Default: 1440 (~2 h).

Default Bonding-Curve Params

Virtual BNB and migration target used when a token creator doesn't override them.

Withdraw Accumulated Fees

Push all collected BNB fees to the current fee recipient.
